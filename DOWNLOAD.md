## Version 1.0 
This is currently the main version of VOS.
VOS is currently only available for the MAC OS Intel-chip platform.
Development for other platforms is still underway!

### System requirements:
    CPU: Intel-core
    RAM: minimum 512MB
    OS: Any MAC OS
    GPU: -
